# Meus Sites Favoritos

Este é um projeto HTML simples desenvolvido como parte de uma atividade escolar. O objetivo é criar uma página web que lista e organiza os sites favoritos do usuário, demonstrando conhecimentos básicos de HTML.

## Funcionalidades

*   **Título da Página:** "Meus sites favoritos"
*   **Título Principal:** "Meus sites favoritos: Diogo Henrique"
*   **Listagem de Sites:** Inclui 5 sites favoritos (Netflix, ChatGPT, YouTube, iLovePDF e Disney+).
*   **Imagens Clicáveis:** Cada site é representado por uma imagem que funciona como um link direto para o site oficial.
*   **Links Descritivos:** Abaixo de cada imagem, o link do site é exibido em texto.
*   **Estrutura Simples:** Utiliza HTML puro e organizado, com comentários explicativos no código.
*   **Responsividade Básica:** Estilização CSS simples para centralizar o conteúdo e garantir uma visualização agradável.

## Tecnologias Utilizadas

*   **HTML5:** Para a estrutura e conteúdo da página.
*   **CSS3:** Para estilização básica e organização visual.
